# Share Secrets

This repo aims at making sharing secrets easy.

[![Build Status](https://travis-ci.org/mathieujobin/share_secrets.svg?branch=master)](https://travis-ci.org/mathieujobin/share_secrets)

## Getting Started

Clone this repo
```bash
bundle install
rake help
```
